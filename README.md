Cryptod
=======

This is a simple Cryptography library written in D.

Ciphers
-------

Block Ciphers:
 * AES
 * Blowfish
 * Threefish

Hashes
------

Context Hashes:
 * SHA1
 
'Fast' Hahses:
 * murmurhash3

PRNGs
-----
 * Mersenne Twister
 * BlumBlumShub
 
More to come.
