Cryptod
=======


This is a simple Cryptography library written in D.

Ciphers
-------

Supports AES, Blowfish, and Threefish so far.

Hashes
------

Supports SHA1 and murmurhash3 so far.

PRNGs
-----

Supports the Mersenne Twister so far.
